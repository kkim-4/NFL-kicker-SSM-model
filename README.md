Kicking is often an undervalued aspect of the NFL football scene with very little analysis considering 25-30% of all the total points in a season consists of extra point attempts and field goal attempts. Most analysis is done retroactively across a singular season and doesn’t help analysts track the trajectories of a kicker’s talent and how it may evolve over time. They also fail in the aspect of often treating kicks as being the same when they are done at completely different distances with varying levels of pressure, however you try to measure pressure.
	To help address this gap in analysis, we introduce FGOE_Pro, an advanced framework built to handle all these different components that will help us build a complete metric to evaluate kickers. This is done by leveraging deep learning and Amortized Variational Inference to build a Bayesian State-Spatial Model to isolate and monitor the evolution of a kicker's latent talent over time.
